# Treatment Voice Scripts

患者役は `morioki / ノーマル`、医師役は `阿井田 茂 / Calm` を前提にしています。
どの台本も、実際の一般外来で違和感が出にくいように、症状確認、リスク確認、生活背景、治療方針、再受診条件までを入れています。

## 1. dm-followup

題材: 2型糖尿病の再診

- 患者: ここ1か月、夕食が遅い日が続いていて、朝の血糖が140台の日が増えています。口の渇きも少しあります。
- 医師: 食後の眠気や、夜中のトイレの回数はどうですか。薬の飲み忘れも確認させてください。
- 患者: 飲み忘れは月に1回あるかないかです。夜中のトイレは1回くらいで、前より少し増えた気がします。
- 医師: 今回の HbA1c は 7.4 で、前回の 6.9 より上がっています。体重も 2 キロ増えているので、食事と運動の影響が大きそうです。
- 患者: 忙しくて歩けていないのと、帰りが遅い日にコンビニで甘い物を足してしまいます。
- 医師: まず夕食後の菓子を週2回までにして、食後10分でも歩きましょう。薬は今のままで、内服の時間だけ固定してください。
- 患者: わかりました。家の血糖と体重をメモして持ってきた方がいいですか。
- 医師: はい、それがあると調整しやすいです。200 を超える値が続く、強い口渇やだるさが出る場合は次回を待たずに連絡してください。

## 2. lipid-first

題材: 脂質異常症の初回治療

- 患者: 健診でコレステロールが高いと言われて来ました。自覚症状はないんですが、放っておいて大丈夫か心配で。
- 医師: 結果では LDL が 178 で、中性脂肪も少し高めです。ご家族に心筋梗塞や脳梗塞の方はいますか。
- 患者: 父が 62 歳で心筋梗塞になっています。私は夜遅くに食べることと、揚げ物が多いです。
- 医師: 家族歴があるので、食事改善だけより薬を併用した方が安全です。筋肉痛が出やすい体質や、肝機能異常の指摘はありますか。
- 患者: それは特にないです。お酒は週に3回、ビールを1本か2本くらいです。
- 医師: では少量のスタチンから始めましょう。揚げ物を減らして、夜食はできれば控えてください。飲酒は量を増やさなければ大丈夫です。
- 患者: 薬はずっと続けることになりますか。副作用が出たらどうしたらいいですか。
- 医師: まず2か月で採血を見ます。筋肉痛や濃い尿が出たら中止して連絡してください。その時点で量の調整や中止を判断します。

## 3. allergic-rhinitis

題材: 花粉症と鼻炎症状

- 患者: 2週間前くらいから、くしゃみと鼻水が止まらなくて、夜も鼻づまりで何回か起きてしまいます。
- 医師: 目のかゆみはありますか。熱や、のどの強い痛みがないかも確認させてください。
- 患者: 目も少しかゆいです。熱はなくて、風邪みたいなだるさはないです。
- 医師: 症状の出方から花粉症が一番考えやすいです。市販薬は使いましたか。
- 患者: 飲みましたが、昼間すごく眠くなってしまって。仕事中につらかったです。
- 医師: 眠気が出にくい内服に替えて、鼻づまりには点鼻薬を併用しましょう。帰宅後に顔を洗うのと、寝る前の洗鼻も役立ちます。
- 患者: 点鼻薬は毎日使っても大丈夫ですか。効くまでどれくらいかかりますか。
- 医師: 用法通りなら問題ありません。数日から1週間で変わってくることが多いので、不十分なら次回さらに調整しましょう。

## 4. gerd-followup

題材: 逆流性食道炎と胃もたれ

- 患者: 食後に胸やけがして、横になると酸っぱいものが上がってくる感じがあります。最近は喉の違和感も少しあります。
- 医師: いつから続いていますか。体重減少、黒い便、飲み込みづらさはありませんか。
- 患者: 1か月くらい前からです。体重は変わらず、黒い便もないです。飲み込みにくさも今のところないです。
- 医師: 強い警戒所見はなさそうです。夕食が遅いことや、コーヒー、脂っこい物は多いですか。
- 患者: 帰宅が遅くて夜10時過ぎに食べることが多いです。コーヒーは1日3杯くらい飲みます。
- 医師: 胃酸を抑える薬を2週間使ってみましょう。食後すぐ横にならないことと、夜食とコーヒーを少し減らすのが大事です。
- 患者: 薬を飲めばすぐ良くなりますか。生活の工夫もかなり効きますか。
- 医師: 数日で楽になることは多いですが、生活面もかなり重要です。改善しない、飲み込みづらい、体重が落ちる場合は胃カメラも検討します。

## 5. acute-cystitis

題材: 急性膀胱炎

- 患者: 昨日から排尿のときにしみる感じがあって、何回もトイレに行きたくなります。下腹部も少し重いです。
- 医師: 発熱や背中の痛みはありますか。尿に血が混じった感じはありませんでしたか。
- 患者: 熱はなくて、背中の痛みもないです。少し濁っている感じはありますが、真っ赤ではないです。
- 医師: 膀胱炎の可能性が高そうです。最近、水分が少ないとか、トイレを我慢することはありましたか。
- 患者: 忙しくてあまり飲めていなくて、外出中は我慢することも多かったです。
- 医師: 尿検査を確認して抗菌薬を出します。今日は水分をしっかり取って、できるだけ我慢しないようにしてください。
- 患者: どれくらいで良くなりますか。仕事は普通にしていて大丈夫ですか。
- 医師: 早ければ1日から2日で楽になります。熱が出る、背中が痛む、吐き気が出る場合は腎盂腎炎のことがあるので早めに受診してください。

## 6. low-back-pain

題材: 腰痛の保存的治療

- 患者: 3日前に重い荷物を運んだあとから腰が痛くて、前かがみになると特に響きます。朝起きたときがいちばん固いです。
- 医師: 足のしびれや力の入りにくさはありますか。排尿や排便の異常はありませんか。
- 患者: しびれはなくて、足にも力は入ります。痛みは腰だけで、お尻から下には広がっていません。
- 医師: 重い神経症状はなさそうです。筋肉や関節まわりの腰痛の可能性が高いです。発熱もないですね。
- 患者: はい、熱はないです。仕事で座りっぱなしなのも関係ありますか。
- 医師: 関係します。痛み止めを数日使って、無理のない範囲で歩いた方が回復は早いです。長時間同じ姿勢は避けてください。
- 患者: 安静にしすぎない方がいいんですね。湿布も使って大丈夫ですか。
- 医師: はい、湿布は併用して構いません。しびれが出る、歩きにくい、排尿排便に異常が出るときはすぐ再受診してください。

## 7. insomnia-stress

題材: 不眠とストレス

- 患者: この1か月くらい寝つきが悪くて、夜中に2回くらい目が覚めます。朝もすっきりしません。
- 医師: 気分の落ち込みや食欲低下はありますか。寝る前の飲酒やスマホの使用も教えてください。
- 患者: 食欲はありますが、疲れやすいです。寝る前にスマホを見て、そのまま缶ビールを飲む日があります。
- 医師: アルコールは寝つきを一時的に良くしても、途中で目が覚めやすくなります。仕事のストレスは最近強いですか。
- 患者: 人手が足りなくて残業が増えていて、頭が休まらない感じがあります。
- 医師: まず起きる時間を固定して、寝る1時間前は画面と飲酒を避けましょう。必要なら短期間だけ使う睡眠薬を少量使う方法もあります。
- 患者: 薬に頼りすぎるのは少し心配です。癖になったりしませんか。
- 医師: 毎日ずっと使う前提ではなく、短期間で様子を見る形です。気分の落ち込みが強くなる、日中の支障が増える場合は早めに相談してください。
