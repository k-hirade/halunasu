# Timed Treatment Voice Scripts

STT と SOAP 生成時間の検証用に、録音時間が段階的に伸びる台本です。
患者役は `morioki / ノーマル`、医師役は `阿井田 茂 / Calm` を想定しています。

実際の読み上げ時間は音声合成の速度、間の取り方、録音環境で変わります。ここでは AivisSpeech の既存設定で、概ね目標時間に近づく文章量を置いています。

## 1. two-minute-uri

目標: 約2分

題材: 発熱と咳の一般外来

- 患者: 3日前から喉が痛くて、昨日から咳と鼻水が出ています。今朝は37度8分まで熱が上がりました。
- 医師: 咳は乾いた咳ですか、それとも痰が絡みますか。息苦しさや胸の痛みも確認させてください。
- 患者: 最初は乾いた咳でしたが、今日は少し黄色っぽい痰が出ます。息苦しさや胸の痛みはありません。
- 医師: 食事や水分は取れていますか。周りに同じような症状の方はいますか。
- 患者: 食欲は少し落ちていますが、水分は取れています。職場で風邪っぽい人が何人かいました。
- 医師: 今のところ肺炎を強く疑う所見はなさそうですが、発熱と咳があるので感染症として対応しましょう。念のため酸素の値と胸の音も確認します。
- 患者: インフルエンザやコロナの検査はした方がいいですか。
- 医師: 発症からの時間を考えると検査はできます。結果で治療方針が変わる可能性があるので、今日は検査をしましょう。
- 患者: 検査の結果が陰性でも、周りにうつす可能性はありますか。家に高齢の母がいるので心配です。
- 医師: 陰性でも一般的な風邪や他のウイルス感染はあります。数日はマスク、手洗い、換気を意識して、食器やタオルは分けましょう。
- 患者: 咳で夜に眠りづらい日があります。横になると少し咳き込みます。
- 医師: 寝る前に水分を少し取り、上半身を少し起こすと楽になることがあります。咳止めは眠前にも使える形で出します。
- 患者: 薬はどんなものになりますか。仕事は休んだ方がいいですか。
- 医師: 解熱鎮痛薬と咳止め、痰を出しやすくする薬を出します。熱がある間は休んでください。息苦しさ、胸痛、39度以上の熱が続く場合は早めに再受診してください。

## 2. three-minute-hypertension

目標: 約3分

題材: 高血圧の再診と生活指導

- 患者: 家で血圧を測ると、朝が150台、夜が140台くらいの日が多いです。薬は毎日飲んでいます。
- 医師: 測る時間は起床後と就寝前で固定できていますか。測る前に喫煙やコーヒーはありますか。
- 患者: 朝は起きてトイレに行ったあとに測っています。コーヒーはその後です。夜は帰宅後すぐ測ることもあります。
- 医師: 夜は少し落ち着いてから測った方が安定します。頭痛、胸の痛み、息切れ、足のむくみはありますか。
- 患者: 頭痛はたまにありますが、強くはないです。胸の痛みや息切れ、むくみはありません。
- 医師: 今日の診察室血圧は148の92です。家庭血圧も高めなので、今の薬だけでは少し足りない可能性があります。
- 患者: 最近、外食が多くて味の濃いものが続いています。運動もあまりできていません。
- 医師: 塩分の影響はありそうです。汁物を残す、漬物や加工食品を減らす、外食では大盛りを避けるところから始めましょう。
- 患者: 薬を増やす必要がありますか。できれば生活で何とかしたい気持ちもあります。
- 医師: 今すぐ大きく増やす必要はありませんが、数値は治療域です。今回は少量の薬を追加して、生活改善も同時に進めるのが安全です。
- 患者: 副作用はありますか。
- 医師: ふらつき、むくみ、動悸が出ることがあります。強い症状があれば中止せず、まず連絡してください。
- 患者: 家の血圧はどう記録すればいいですか。
- 医師: 朝と夜に2回ずつ測って、平均に近い値をメモしてください。次回4週間後に手帳を見て、薬の量を調整しましょう。
- 患者: 食事は朝昼夜で、どこから変えるのが一番いいですか。
- 医師: まず朝食を抜かないことと、昼の麺類や丼ものを減らすことです。野菜やたんぱく質を先に食べる形にしましょう。
- 患者: 仕事中はついカップ麺やコンビニ弁当が多くなります。
- 医師: カップ麺は汁を残しても塩分が多めです。選ぶならサラダ、ゆで卵、焼き魚、豆腐などを足して、味の濃いおかずを減らしましょう。
- 患者: 睡眠時間も関係しますか。最近は5時間くらいの日が多いです。
- 医師: 睡眠不足は血圧を上げやすくします。まず就寝時刻を30分だけ早める日を週に数回作るところから始めましょう。
- 患者: 目標の血圧はどれくらいですか。
- 医師: 家庭血圧ではまず135未満を目指します。急に下げるより、記録を見ながら安全に調整します。
- 患者: わかりました。急に病院に来た方がいい症状はありますか。
- 医師: 強い胸痛、片側の手足の動かしにくさ、ろれつが回らない、激しい頭痛があれば救急受診してください。

## 3. four-minute-abdominal-pain

目標: 約4分

題材: 腹痛と下痢の初診

- 患者: 昨日の夜からお腹が痛くて、下痢が5回くらいあります。今朝も水っぽい便が出ました。
- 医師: 痛みはお腹のどのあたりですか。ずっと痛いのか、波があるのかも教えてください。
- 患者: おへその周りから下腹部にかけてです。ずっと重い感じがあって、時々きゅっと痛みます。
- 医師: 吐き気や嘔吐、発熱はありますか。便に血が混じる感じはありませんか。
- 患者: 吐き気は少しありますが、吐いてはいません。熱は37度3分くらいで、血は見えていません。
- 医師: 昨日から今日にかけて、生ものや普段と違う食事はありましたか。周囲に同じ症状の人はいますか。
- 患者: 昨日の昼に鶏肉の定食を食べました。家族は同じものを食べていないので、周りに同じ症状はいません。
- 医師: 水分は取れていますか。尿の回数が減った、口が乾く、立つとふらつく感じはありますか。
- 患者: 水は飲めていますが、食事はあまり取れていません。尿は少し少ない気がします。立つと少しふらっとします。
- 医師: 脱水が少しありそうです。お腹を診察して、強い圧痛や反跳痛がないか確認します。今の話では感染性胃腸炎が考えやすいですが、虫垂炎なども見逃さないようにします。
- 患者: 検査は必要ですか。
- 医師: 強い腹膜刺激症状がなければ、今日は血液検査までは必須ではありません。ただ、痛みが右下腹部に移る、熱が上がる、血便が出る場合は検査が必要です。
- 患者: 薬は下痢止めを飲んでもいいですか。
- 医師: 血便や高熱がある時は強い下痢止めは避けます。今回は整腸剤と吐き気止めを中心にして、脱水予防を優先しましょう。
- 患者: 食事はどうすればいいですか。
- 医師: 今日は経口補水液や薄い味噌汁などで水分と塩分を取ってください。食べられるようなら、おかゆ、うどん、バナナなど消化の良いものにしましょう。
- 患者: 仕事は行かない方がいいですか。
- 医師: 下痢が続いている間は無理しないでください。食品を扱う仕事なら特に休んだ方が安全です。
- 患者: 市販の痛み止めは飲んでもいいですか。
- 医師: 胃腸が弱っている時は薬で悪化することがあります。今回は自己判断で強い痛み止めを足さず、処方薬を使って様子を見てください。
- 患者: 家族にうつる可能性はありますか。小さい子どもがいます。
- 医師: 感染性胃腸炎ならうつることがあります。トイレ後の手洗い、タオルの共有を避けること、便で汚れた場所の消毒を意識してください。
- 患者: 水分は一度にたくさん飲んだ方がいいですか。
- 医師: 一度に多く飲むと吐き気が強くなることがあります。少量をこまめに、5分から10分おきに飲む方が続けやすいです。
- 患者: 痛みが右下に移るというのは、どんな感じなら危ないですか。
- 医師: 右下腹部を押すと強く痛む、歩くと響く、咳でお腹に響く、熱が上がる場合は虫垂炎なども考えるので早めに連絡してください。
- 患者: 明日も下痢が続いたら、すぐ来た方がいいですか。
- 医師: 回数が減って水分が取れていれば様子を見られます。悪化する、尿が出ない、血便がある、強い腹痛がある時は早めに受診してください。
- 患者: 吐き気が強くなって薬が飲めない場合はどうしたらいいですか。
- 医師: 水分も薬も入らない状態が続くなら点滴が必要になることがあります。半日以上続く時は我慢せず連絡してください。
- 患者: どのくらいで良くなりますか。
- 医師: 多くは1日から3日で改善します。水分が取れない、尿が半日以上出ない、強い腹痛、血便、38度以上の熱が続く場合は早めに受診してください。

## 4. five-minute-diabetes-complex-followup

目標: 約5分

題材: 糖尿病、高血圧、脂質異常症の再診

- 患者: 今日は糖尿病の定期受診です。最近、朝の血糖が少し高くて、140から160くらいの日があります。
- 医師: 食後の血糖や、低血糖のような冷や汗、手の震え、強い空腹感はありましたか。
- 患者: 低血糖っぽい感じはありません。食後はあまり測れていませんが、夕食が遅い日の翌朝は高いです。
- 医師: 内服は毎日飲めていますか。飲む時間がずれることはありますか。
- 患者: 飲み忘れはほとんどありません。ただ、夜の薬は帰宅が遅くなると深夜近くになることがあります。
- 医師: 今回の HbA1c は7.3です。前回が7.0だったので少し上がっています。体重は1.5キロ増えています。
- 患者: 年度末で残業が増えて、夕食が22時過ぎになる日が多かったです。コンビニで麺類とおにぎりを一緒に買うこともあります。
- 医師: 炭水化物が重なっているのと、食事時間が遅い影響がありそうです。運動はできていますか。
- 患者: 以前は週に2回歩いていましたが、最近はほとんどできていません。休日も疲れて寝てしまいます。
- 医師: 血圧手帳を見ると、朝が145前後で、夜は135前後ですね。塩分が多い食事や外食も増えていますか。
- 患者: はい、ラーメンや丼ものが多いです。汁は残すようにしていますが、完全にはできていません。
- 医師: LDLコレステロールは112で、前回より少し下がっています。脂質の薬は効いています。筋肉痛や濃い尿はありませんか。
- 患者: それはありません。薬の副作用らしいものは特に感じていません。
- 医師: 目のかすみ、足のしびれ、傷が治りにくい感じはありますか。
- 患者: 目のかすみは疲れた時に少しあります。足のしびれはありません。傷も特にないです。
- 医師: 眼科の糖尿病チェックはいつ受けましたか。
- 患者: 半年前くらいです。次はまだ予約していません。
- 医師: 年1回は必要なので、次回までに予約しておきましょう。腎機能と尿検査は今日確認します。
- 患者: 薬は増えますか。できれば増やしたくないですが、数値が悪いなら仕方ないと思っています。
- 医師: いきなり大きく増やすより、まず夕食の炭水化物を重ねないこと、食後10分歩くこと、夜の薬の時間を固定することを優先しましょう。ただし血圧は少し高いので、降圧薬は少量調整します。
- 患者: 糖尿病の薬は今回はそのままですか。
- 医師: はい、今回は糖尿病薬はそのままにします。4週間から6週間で生活の変化を見て、HbA1cの流れで追加を考えます。
- 患者: 食事で具体的に何を変えるのが一番効果的ですか。
- 医師: 夜遅い日は、麺とおにぎりを両方にしないことです。麺ならおにぎりをやめる、おにぎりならサラダチキンや豆腐を足す形にしましょう。
- 患者: 運動はどのくらい必要ですか。
- 医師: まず毎日でなくていいです。夕食後に10分歩く日を週4回作りましょう。雨の日は室内で足踏みでも構いません。
- 患者: 家で測るものは血糖と血圧の両方ですか。
- 医師: 可能なら朝の血糖を週3回、血圧は朝晩で記録してください。全部できなくても、続けられる範囲で大丈夫です。
- 患者: 仕事が忙しい時に全部やろうとすると続かない気がします。
- 医師: 完璧にやる必要はありません。まず血圧は週3日、血糖も週3回で構いません。記録が空いた日があっても再開できれば十分です。
- 患者: 夜の薬を飲む時間は、夕食が遅い日でも同じ方がいいですか。
- 医師: できるだけ時間をそろえます。夕食がかなり遅い日は、薬の種類によって調整が必要なので、今日の処方に合わせた飲み方を紙に書いて渡します。
- 患者: 次回までの目標は何を一番優先すればいいですか。
- 医師: 夜の炭水化物を重ねないこと、食後10分歩くこと、血圧と血糖を週3回以上記録することです。次回はその3点を見て薬を調整します。
- 患者: 受診を早めた方がいい時はありますか。
- 医師: 血糖が250を超える日が続く、強い口渇や体重減少がある、胸痛や息切れがある、血圧が180を超えて頭痛が強い場合は早めに連絡してください。
- 患者: わかりました。次回はいつ来ればいいですか。
- 医師: 6週間後にしましょう。採血と尿検査の結果を見て、糖尿病薬と血圧薬の調整を決めます。
