# v2 カルテ文体仕様

目的: 実際の電子カルテSOAP記載に近い文体で書く。v1の「採点情報を含む説明文」を排除する。

## 必須(リント対象)

1. **メタ文の禁止**: 「当日確認した主な診療内容は」「確認すべき論点は」「点数に直結する名称」「会計前に」等の採点メタ文を含まない。
2. **点数表正式名称の禁止**: 全角の正式名称(ＣＴ撮影（１６列以上…）、処方箋料（リフィル以外・その他）等)を本文に書かない。例外: 薬剤名・検査略称など臨床で実際に使う名称(ゲーベンクリーム、HbA1c、インフル迅速)は可。
3. **施設属性の禁止**: CT列数・電子画像管理体制・施設基準届出をカルテに書かない(施設fixtureが持つ)。
4. **実施根拠の述語**: 当日実施は「施行/実施/行った/(検査名)提出」で書く。「〜を確認した」だけを実施根拠にしない。
5. **数値根拠**: 面積・量・日数が点数区分に効くものは数値を書く(「約8×6cm」「10g」「7日分」)。

## 推奨(リアリズム軸 — ケースごとに2つ以上)

- 略語: BT/KT, BP, P, SpO2, p.o., Tab, Cap, ED(各科慣用)
- 体言止め・短文: 「咽頭発赤(+)。扁桃白苔なし。」
- 検査結果の生値: 「WBC 9800, CRP 2.1」
- 陰性所見・否定: 「嘔吐なし、髄膜刺激徴候なし」
- 過去・外部情報の混在: 「前医でセフェム処方済み」「健診でHbA1c 6.9指摘」(時制・帰属が分かる書き方で)
- 予定の混在: 「改善なければ次回採血予定」
- 患者の言葉: 「『夜眠れないくらい痒い』」
- 表記揺れ: 半角/全角、カナ薬剤名、「インフル」「溶連菌」等の略称

## 禁止されるリアリズム

- 実在患者由来の文章の転記(文体参考のみ可)
- 根拠の曖昧化が答えを変えるレベルの省略(exactケースでは期待コードの根拠文は必ず特定可能に書く)

## 長さ

S/O/A/P合計の目安(難易度別):

- L1(雑情報少なめ): 400〜600字
- L2(ディストラクタ混在): 550〜900字
- L3(時系列・帰属の混在が濃い): 700〜1100字

v1(平均1064字の説明調)とは異なり、行数は多く1行は短い実カルテの密度で書く。各ケースは distractors(本文に存在するが算定してはならない情報)を難易度に応じて2〜4件含める。
