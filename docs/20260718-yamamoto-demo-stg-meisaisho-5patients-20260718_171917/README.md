## 評価CLIの追随修正

初回実行では、各セッションに算定line itemがある一方、患者月次が全件0点になった。現行仕様では
line itemの初期状態は未承認で、月次集計は未承認行を合計へ含めない。従来の評価CLIは
`POST /calculate`後に採否を保存せず、そのまま月次を取得していた。

製品側の安全仕様は変更せず、評価CLIへ`--approve-calculated-lines`を追加した。このオプションは
次のline itemだけを明示承認する。

- `sourceType=line_item`
- workspaceで非表示ではない
- `excludedFromTotal`ではない
- statusが`blocked`または`rejected`ではない

候補提案（candidate proposal）やblocked行は自動承認しない。今回の最終5患者測定では、この条件で
30 line itemを承認・保存してから月次を取得した。
