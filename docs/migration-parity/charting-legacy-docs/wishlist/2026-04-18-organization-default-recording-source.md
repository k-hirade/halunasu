# Wish List: 病院全体の既定録音方法

Status: Backlog

## Context

2026-04-18 時点では、録音方法の既定値はメンバーごとに保持する。診療セッション作成時は担当メンバーの既定値を初期値にし、セッション内では録音開始前に変更できる。

将来的に、病院全体で「新規メンバーの初期値」または「メンバー未設定時のフォールバック」を管理したい可能性がある。

## Goal

病院設定として既定の録音方法を持ち、メンバー設定と矛盾しない優先順位で診療セッションの初期値を決定できるようにする。

## Proposed Behavior

- 病院設定に `defaultRecordingSource` を追加する。
- メンバーに個別設定がある場合は、メンバー設定を優先する。
- メンバーに個別設定がない場合は、病院設定を使う。
- 病院設定も未設定の場合は、現行の既定値 `linked_mobile` を使う。
- 新規メンバー作成時の初期値として病院設定を利用するかは、実装時に運用要件を確認する。

## Non-Goals

- 既存メンバーの設定を病院設定で一括上書きすること。
- 録音開始済みまたは書き起こし済みセッションの録音方法を変更すること。

## Acceptance Criteria

- 病院管理者が設定画面で病院全体の既定録音方法を変更できる。
- セッション作成時の優先順位が `メンバー設定 > 病院設定 > システム既定値` になっている。
- 既存メンバーの個別設定は、病院設定変更後も維持される。
- 監査ログに病院設定変更が残る。
