# 責任分界

更新日: 2026-05-06

## 目的

本書は、ハルナスと利用医療機関等の責任分界を整理するための内部資料である。対外説明資料や医療機関向けセキュリティチェックシートの回答の基礎として用いる。

## ハルナスの責任

- アプリケーションの提供、保守、障害対応
- アカウント認証、権限管理、監査ログの提供
- 診療音声、文字起こし、SOAP 下書きの処理基盤の運用
- 外部委託先の選定、契約、監督
- データの保存期間管理、削除処理、インシデント時の委託先対応
- 医療機関に対するインシデント通知と技術的支援

## 利用医療機関等の責任

- 患者本人への説明と必要な同意取得
- 録音実施の判断と運用
- 本サービスで生成された SOAP 下書きの確認、編集、確定
- 診療上の判断
- 法令上保存義務のある正式な診療録の作成・保存
- 院内端末、ネットワーク、職員アカウントの運用管理
- 退職者や異動者の権限見直し

## 責任分界上の重要な前提

- ハルナスが生成する SOAP は診療録の下書きであり、正式診療録そのものではない
- 録音・診療・同意・診療録保存は医療機関等が主体となる
- ハルナスは委託先として必要な安全管理措置を講じる
