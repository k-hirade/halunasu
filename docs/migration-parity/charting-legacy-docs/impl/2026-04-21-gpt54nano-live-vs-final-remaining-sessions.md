## 読み方

- 速度はOpenAI API開始から完了までの壁時計時間
- 初回出力秒はストリーミングで最初に `output_text` が取れた時刻
- 精度は出力全文を見て、live入力とfinal入力で重要情報の抜け・誤解釈・薬剤/検査/再診指示の差を確認する
- final source が `live_stt_fallback_short_final` の場合、最終STTとして再利用された内容が実質live寄りである点に注意する
