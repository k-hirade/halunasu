## 判断

今回のデータ範囲では、在宅診療料、在宅系加算、がん性疼痛緩和指導管理料、文書料を含め、構造化ordersからの決定論的な再算定は完全再現できた。

ただし、これは合成データ3患者と確定コード付きordersの再現試験であり、カルテ本文からのOpenAI抽出精度や実病院データ全体の正解率を示すものではない。既存レセを医学的・請求上の正解と保証する試験でもない。

対象3患者の `unknowns.csv` には合計43行が残る。薬剤コード・数量、診療日単位の病名、完全一致しないマスタ項目などは今回の15コードに含まれず、完全一致の分母外である。

## 残る改善

1. 診療月がマスタ適用期間外の場合に、0点で静かに完了せず明示警告を返す。
2. マスタ未解決時の表示へ内部の具体的な `reason` を反映する。
3. 西山病院の実データでは、対象月に対応するマスタ世代を固定してから比較する。
4. カルテ本文からの抽出を含む精度は、通常の臨床抽出経路を通す別試験で評価する。
5. `engineCoverageDecision.homeCare` は一致結果でも「在宅系は当社未対応」と返しており、実結果と矛盾するため診断文を動的な判定へ修正する。
